# Stagger
Hi! Stagger is a rhythm game.

# Building
Stagger is built using Godot Engine 4.7.

Here's some GUI instructions because I dunno how to do it with CLI.
- Open the project.
- In the top menu, go to `Project > Export...`
- If there are no predefined exports, add one for the target system.
	- You must add an export path for all exports you add.
- Click `Export All...`
- Wait
